# Experiments

This directory contains scripts used to **analyze model outputs and generate the figures used in the final project presentation**.  
The scripts operate on experiment outputs produced by the training and evaluation pipeline located in:

```
student/final_project/arithmetic_llm/
```

These scripts perform tasks such as:

- computing adversarial robustness metrics
- generating baseline comparison tables
- creating visualizations of experiment results
- producing artifacts used in the final presentation

All outputs generated by these scripts are saved to the repository's:

```
results/
```

directory.
---

# Script Overview

## analyze_adversarial_results.py

Analyzes adversarial evaluation outputs and computes robustness metrics such as:

- answer flip rates
- incorrect revision patterns
- adversarial sensitivity across perturbation types

This script produces summarized metrics used to evaluate model robustness.

---

## generate_baseline_table.py

Generates the baseline results table used to compare model performance before and after adversarial perturbations.

The output is written to:

```
results/comparison_results.csv
```

---

## generate_presentation_artifacts.py

Produces figures and artifacts used directly in the project presentation, including:

- accuracy comparison plots
- adversarial robustness visualizations
- aggregated experiment summaries

All generated plots are saved to:

```
results/
```

---

## generate_additional_presentation_plots.py

Creates supplementary visualizations used for deeper analysis of experiment results.  
These plots help illustrate patterns in adversarial behavior and model response dynamics.

---

## generate_cross_axis_wrong_revision_heatmaps.py

Generates heatmaps that visualize incorrect revision patterns across different experiment conditions.  
These plots highlight how adversarial prompts influence the model's reasoning process.

---

## split_baseline_adv_def_grouped_plot.py

Creates grouped plots comparing:

- baseline predictions
- adversarially perturbed predictions
- defensive model behavior

These visualizations help illustrate how model responses change under adversarial influence.

---

## split_cross_axis_grouped_plots.py

Produces grouped visualizations across different experimental axes, allowing comparison of performance under multiple perturbation scenarios.

---

## split_cross_axis_grouped_plots_strict.py

A stricter version of the grouped cross-axis visualization that applies tighter filtering criteria when analyzing model response changes.

---

## split_cross_axis_wrong_revision_faceted_strict.py

Creates faceted plots highlighting incorrect revision patterns under strict filtering conditions.  
These plots help isolate specific adversarial failure modes.

---

# Running Experiment Analysis

All scripts should be executed from the **repository root**.

Example:

```bash
python experiments/analyze_adversarial_results.py
```

Generated tables and figures will be written to:

```
results/
```
